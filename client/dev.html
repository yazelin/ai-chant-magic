開發工具箱

真。AI。咏唱魔法 — 開發期用的工具集中在這。改東西 / 要驗收,就用對應工具聽或看,給真實回饋。

🎮 遊戲 / 首頁練習場

正式遊戲。首頁就是詠唱練習場(四角色卡 + 麥克風練喊 + 技能詠唱詞可改)。

🔊 音效實驗室

直接聽程序合成的 SFX(事件表)與自適應背景音樂(四首隨波數切換),不用開遊戲。

🎨 角色 Sprite 挑選器(sprite-forge)

用一張 AI 圖抽大量姿勢 → 瀏覽器挑 → 組成 walk/idle/cast sprite。四角色造型都用它做的。

需另開伺服器:終端跑 tools/sprite-forge/forge.sh serve,再開 http://127.0.0.1:8799/picker.html?job=<job>(SOP 見該夾 README)

訓練假人(元素反應驗證)

單機開局,身邊放一隻不會動、打不死的假人,可直接灌任一元素殘留 → 用自己職業的招式當第二擊,真實觸發反應(外環/爆發特效/音效/wavehud 反應計數都是正式遊戲的程式碼)。

加 &class=cryo(或 storm/warden)選職業;只在 npm run dev 開發模式生效,不進正式部署。
註:音效實驗室是 Vite dev 頁(npm run dev:client),不會進正式部署。
